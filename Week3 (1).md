# 1️⃣ 개념정리

## 01. 결과와 평가는 다르다

```
✅ 학습 목표 :
* 수치로 나타난 결과와 그에 대한 평가는 구분되어야 함을 이해한다.
* 결과를 해석할 때 반드시 목적과 기준을 명화히 해야 함을 인식한다. 
```
사실과 결과의 데이터만을 그래프, 표, 지표 등으로 표시하는것과

내용을 평가해서 구체적인 행동과 판단으로 연결하는 것은 

**다르다!!**




## 02. 비교할 때는 관점이 중요하다

```
✅ 학습 목표 :
* 같은 데이터라도 해석 관점에 따라 전혀 다른 메시지가 도출될 수 있음을 이해한다.
* 비교 대상, 비교 기준, 목적을 명확히 해야 의미 있는 비교가 가능함을 설명할 수 있다. 
```
- 데이터 평가는 비교 대상이 있어야 가능하다.숫자만 보고 크고 작음을 따지는 건 ‘인상’에 불과하다.
- 결론을 먼저 정하고 데이터를 끼워 맞추면 왜곡이 생긴다.
- 데이터는 객관성과 논리성에 기반해 해석해야 한다.
- 결과’ = 단순 사실, ‘결론’ = 그 사실의 의미.
- 비교할 때 두 가지 체크포인트:
  - ① 결론으로 이어지는 결과인가?
  - ② 차이를 찾을 수 있는가?
- 비교 방식: 내부 비교(자사 내), 외부 비교(다른 산업/경쟁사).
- 차이가 없어도 잘못된 게 아니라, 가설이 틀렸을 수도 있음.



## 03. 비교의 기술

```
✅ 학습 목표 :
* 데이터 비교에서 편차, 표준편차의 사용되는 과정을 이해한다.
* 지표의 선택과 해석이 결과에 미치는 영향을 판단할 수 있다. 
```
단순히 평균값만 비교하면 데이터의 본질을 놓칠 수 있다.
- 예: 상품 C의 평균 매출이 가장 높지만, 월별 추이를 보면 일시적 요인으로 높은 경우가 있음.
- 따라서 평균뿐 아니라 추이와 변화를 함께 분석해야 한다.
- 매출 변동성을 보기 위해서는 **편차(표준편차, 분포도)**를 고려하는 것이 중요하다.
- 상품 C는 평균은 높지만 편차도 가장 커서 안정적이라고 보기 어렵다.
- ‘편차가 크다’ = 변동성이 크다는 뜻이지만, 반드시 ‘나쁘다’는 의미는 아니다.
- 해석에 따라 위험 요인이 될 수도, 성장 기회가 될 수도 있다.
- 데이터 해석은 좋고 나쁨을 단정하기보다 목적에 맞게 결론을 도출해야 한다.
- 평가 기준에는 값의 크기, 추이, 분포 폭, 비율 등이 있으며, 상황에 맞게 복합적으로 활용해야 한다.
- 같은 데이터라도 어떤 기준으로 비교하느냐에 따라 결론이 달라질 수 있다. 단순히 평균값만 비교하면 데이터의 본질을 놓칠 수 있다.
- 예: 상품 C의 평균 매출이 가장 높지만, 월별 추이를 보면 일시적 요인으로 높은 경우가 있음.
따라서 평균뿐 아니라 추이와 변화를 함께 분석해야 한다.


## 04. 비교 사례 분석 

~~~
✅ 학습 목표 :
* 실제 사례를 통해 데이터 비교에서 발생할 수 있는 상황들을 식별한다.
* 다양한 맥락에서 의미 있는 비교를 하기 위한 분석적 사고력을 기른다. 
~~~


- 비교 대상 설정이 핵심: 무엇과 비교하느냐에 따라 결론 달라짐.
 - 비교 방법:
  - ① 인접 구, ② 규모 비슷한 구, ③ 증가율(비율), ④ 인구밀도, ⑤ 내부 지역 간 비교.
	•	메구로구 사례:
	•	인접 구와 비교 → 하위권.
	•	규모 비슷한 구와 비교 → 다른 결론.
	•	비율·밀도 기준 → 강점도 드러남.
	•	내부 비교 → 세부 지역 차이 확인 가능.

| 한강 공원       | 4.6           | 1,200       | 1.4                      |

>
> **위 사례에서 예린이의 접근 방식은 데이터 문해력 관점에서 타당한지, 각 장소의 평점에 담긴 의미를 활용해 비교해보세요.**

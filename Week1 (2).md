# 1️⃣ 개념정리
## 01. 기계가 할 일과 사람이 할 일

```
방대한 데이터 사회를 살아가는 우리에게 데이터를 다루고 특징을 도출해낼 수 있는 기술은 매우 중요한 것으로 여겨진다. 이를 위해서 우리는 통계학 지식, 분석 이론 및 방법, 데이터 가공 및 분석 작업 방식 등을 배운다.
그러나 과연 이 기술들 중 기계보다 사람이 더 잘할 수 있는 것은 얼마나 될까? 이러한 기술들은 사실 기계의 발전에 따라 기계가 훨씬 더 빠르고 정확하게 해낼 수 있을 것이다. 
그러므로 우리 인간은 기계에게 맡길 수 없는, 오직 사람만이 할 수 있는 것은 무엇인지 구별하고, 이 기술과 가치에 대한 깊은 이해가 필요하다.
```

## 02. 통계를 배워도 왜 활용하지 못하나

```
✅ 학습 목표 :
아무리 훌륭한 통계 지식이 있어도 목적에 따라 필요한 정보나 목표를 주지 않으면 효과적으로 동작할 수 없다. 데이터 활용은 3가지 단계로 이루어진다. (1)분석 전 문제 정의/가설 구축, (2)분석을 위한 기술과 지식, (3)분석 결과에 대한 해석 및 스토리 구축이다. 그중 (2)는 계산과 분석의 결과로, 인간보다 기계가 더 우수하다. 따라서 문제가 무엇인지 파악하고, 분석 결과를 토대로 결론을 내리는 것이 가장 중요하다는 것이다.
(1)과 (3)은 소위 말하는 데이터 문해력 기술, 데이터 활용을 잘 못하는 본질적인 이유가 될 수 있는 고부가가치 기술이다. 앞으로 이 책에서는 이 기술에 대해 깊이 파고들 것이다.
```

## 03. 데이터를 먼저 보지 마라 - 데이터 안에 답은 없다
```
데이터 활용을 못하는 사람들은 모두 공통점이 있다. 그건 바로 데이터 안에 정답이 있다는 생각이다. 데이터를 아무리 바라보고, 다양한 분석기법을 사용하며 그래프를 도출해낸다한들 그 목적이 명확하지 않으면 이는 큰 의미가 없다는 것이다. 
데이터 분석 그 자체가 목적이 되면 안된다. 대신, 이는 자신의 목적을 충족시키기 위해 사용하는 도구로 받아들여야 한다. 
책에서 말하는 데이터 활용 프로세스는 다음과 같다: 겉으로 드러난 현상 -> 목적 및 문제 정의 -> 지표 결정 -> 현재 상태 파악 -> 평가 -> 요인 분석 -> 해결 방안 모색 
```
